Integration Method

This page uses the automatic initialization method. Simply add the data-layout-auto-init attribute to the <body> tag: 

<body data-layout-auto-init>
  <!-- Your page content -->
</body>

<!-- Include the layout loader script -->
<script src="assets/js/layout-loader.js"></script>

The layout loader will automatically fetch and inject the header and footer components when the DOM is ready, then initialize all interactive elements.

Manual Initialization

For more control, you can manually initialize the layout loader: 

<script src="assets/js/layout-loader.js"></script>
<script>
  // Wait for DOM ready
  if (document.readyState === 'loading') {
    document.addEventListener('DOMContentLoaded', initLayout);
  } else {
    initLayout();
  }

  async function initLayout() {
    const result = await window.LayoutLoader.init();
    
    if (result.success) {
      console.log('‚úÖ Layout loaded successfully');
    } else {
      console.error('‚ùå Failed to load layout', result);
    }
  }
</script>

Testing Checklist

Verify the following functionality:

  • Header appears at the top of the page
  • Footer appears at the bottom of the page
  • Hamburger menu opens when clicked
  • Menu closes with close button, overlay click, or Escape key
  • Language selector opens and allows language selection
  • Dark mode toggle works and persists in localStorage
  • TTS toggle works and persists in localStorage
  • All links navigate correctly
  • Keyboard navigation works (Tab, Enter, Escape)
  • Responsive design works on mobile, tablet, and desktop
