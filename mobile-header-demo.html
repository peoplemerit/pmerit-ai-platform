Integration Instructions

To use the mobile header in your pages:

  1. Include the CSS: <link rel="stylesheet" href="assets/css/clean-mobile.css">
  2. Include the JS: <script src="assets/js/clean-mobile.js"></script>
  3. Add the HTML component at the top of your body tag
  4. Ensure Font Awesome is loaded for icons
  5. The header will automatically show on mobile (<768px) and hide on desktop
