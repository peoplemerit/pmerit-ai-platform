üìê Color Standardization Reference

This section documents the standardized color system across the PMERIT platform. All pages should use these CSS variables for consistency.

Light Mode Color Mapping

Variable Hex Value Usage
--color-primary #375b8d Main brand blue - headers, primary text on light
--color-secondary #4AA4B9 Brand teal - secondary accents
--interactive-primary #375b8d Primary buttons background
--interactive-primary-hover #2a4a73 Primary buttons hover state
--bg-gradient-start #375b8d Hero gradient start (left/top)
--bg-gradient-end #4AA4B9 Hero gradient end (right/bottom)

Dark Mode Color Mapping

Variable Hex Value Usage
--color-primary #598bca Lightened brand blue for readability
--color-secondary #5FC8E5 Lightened teal for dark backgrounds
--interactive-primary #598bca Primary buttons - LIGHTER for visibility
--interactive-primary-hover #6a9bd8 Primary buttons hover - gets lighter
--bg-gradient-start #1E3A5F Hero gradient start - DARKER for depth
--bg-gradient-end #2A4A6A Hero gradient end - DARKER for depth
‚ö†Ô∏è Important Design Decision:
In dark mode, hero gradients use DARKER colors (#1E3A5F ‚Üí #2A4A6A) while buttons use LIGHTER colors (#598bca). This is intentional:
  • Dark hero gradients provide visual depth and contrast
  • Light buttons ensure visibility and clickability
  • This matches the design of career.html (reference page)

üìã CSS Variable Quick Reference

Copy-paste ready variable names for common use cases

For Buttons

.btn-primary {
    background: var(--interactive-primary);
    color: var(--text-inverse);
}
.btn-primary:hover {
    background: var(--interactive-primary-hover);
}

For Hero Sections

.hero {
    background: var(--gradient-hero);
    /* OR */
    background: linear-gradient(135deg, var(--bg-gradient-start), var(--bg-gradient-end));
}

For Cards

.card {
    background: var(--bg-card);
    border: 1px solid var(--border-color);
    color: var(--text-primary);
}

For Links

a {
    color: var(--text-link);
}
/* Note: In dark mode, --text-link is coral (#FF7B7B) */
